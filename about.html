Meet Our Team

We’re passionate about creating amazing web experiences. Here’s a little bit about each of us.

Rishitha

FrontEnd Developer

The frontend developer is responsible for building the visual components of the web application that users interact with. They ensure the interface is responsive, user-friendly, and integrates smoothly with backend services.

Yavanica

Project Manager

The project manager oversees task allocation, timeline tracking, and coordination among team members. As a QA tester, they also test the application thoroughly to detect bugs and ensure a high-quality, bug-free user experience.

Shrika

DataBase Administrator

The DBA designs, manages, and maintains the database system to ensure reliable and secure data storage. They handle data organization, backups, performance tuning, and ensure data integrity across the platform.

Namrata

BackEnd Developer

The backend developer handles the server-side logic, manages the database, and develops APIs that connect the frontend to the backend. They ensure data is processed, stored, and retrieved efficiently and securely.

Manasvi

UI/UX designer

The UI/UX designer focuses on creating an intuitive and engaging user experience. They design wireframes and prototypes, keeping usability and accessibility in mind, to ensure a smooth customer journey throughout the app.
